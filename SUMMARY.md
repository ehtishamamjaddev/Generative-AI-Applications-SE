## 🛠️ TECHNOLOGY STACK

### No-Code Automation
- **n8n** — Visual workflow engine, webhooks, scheduling
- **Integration:** Google Sheets, Google Drive, external APIs

### Backend Framework
- **FastAPI** — Modern Python web framework with OpenAPI docs
- **Architecture:** RESTful APIs with agentic capabilities

### LLM & Embeddings
- **Groq LLM** — llama3-8b8192, free tier, fast inference
- **Embeddings:** BAAI/bge-small-en-v1.5 (768 dimensions)
- **Local Embeddings:** Ollama for privacy-preserving alternatives

### Vector Database
- **Pinecone** — Managed vector search, free tier
- **Chunking:** 512 tokens with 50-token overlap
- **Namespaces:** Separate for each domain (finance, healthcare, research)

### External APIs
- **Google Cloud:** Sheets API, Drive API, OAuth2
- **Tavily:** Web search integration
- **Wikipedia:** REST API for general knowledge

### Frontend
- **HTML5 + JavaScript** — Interactive chatbot widget
- **Features:** Tool trace display, citation rendering, multi-turn chat
